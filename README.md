# Warpgate Connect

## Installation

### Linux / macOS

Run `curl -fsSL https://raw.githubusercontent.com/Stax124/warpgate-connect/refs/heads/main/install.sh | sh`

### Windows

1. Install Rufus from https://rufus.ie/en/
2. Download CachyOS ISO from https://cachyos.org/
3. Find a USB drive and put it into your laptop/computer
4. Open Rufus
5. Select the USB as a target
6. Select the ISO that you have downloaded
7. Click "Start" and wait for the process to finish
8. Reboot your device and select the USB drive as boot drive
9. Install Linux
10. Be happy
